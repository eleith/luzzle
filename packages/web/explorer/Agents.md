# Agents.md for @luzzle/web

Instructions for agents working on the `@luzzle/web` package (Web Explorer).

## Scope

The Web Explorer is a read-only viewer for Luzzle archives, typically deployed
via Docker.

## Architecture

SvelteKit application.

- **Config:** Managed by `@luzzle/web.tools` via `config.yaml`. Controls theme,
  database location, and piece mapping.
- **Data Source:** Reads from the SQLite database generated by
  `@luzzle/web.tools`.
- **Rendering:** Server-Side Rendering (SSR) via `@sveltejs/adapter-node`.

## Key Concepts

- **Docker Deployment:** Users build custom images using
  `examples/Dockerfile.custom` to bundle content.
- **Read-Only:** Strictly reads data; does not modify the archive.
- **Database:** Superset of the core database. `@luzzle/web.tools` adds tables
  for web aggregation and full-text search.

## Development

- **Dev Server:** `npm run dev`
- **Build:** `npm run build`
- **Check:** `npm run check`
